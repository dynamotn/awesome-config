Feature
=======

* Autostart application
* Quake drop-down terminal

Installation
=======

```
cd ~
git clone https://github.com/dynamotn/awesomewm-config
ln -s ~/awesomewm-config $XDG_CONFIG_HOME/awesome
cd awesomewm-config
git submodule update --init --recursive
```
